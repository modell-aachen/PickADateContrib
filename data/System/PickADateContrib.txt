---++ Usage

This contrib introduces the formfields =date2= and =time2= which are used to
render a date or time picker control based on
[[http://amsul.ca/pickadate.js/][pickadate.js]].

For further documentation and API usage see:
   * [[http://amsul.ca/pickadate.js/date/][datepicker]]
   * [[http://amsul.ca/pickadate.js/time/][timepicker]]
   * [[http://amsul.ca/pickadate.js/api/][api]]

---++ Developer Info

Values for field =date2= are stored as Unix epoch. Values for field =time2= as
minutes starting from 00:00am.
